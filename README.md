# Treatment Advice Review Site

This folder contains a standalone review site for the 50 selected treatment-advice samples.

## Files

- `index.html`: main review page
- `styles.css`: page styles
- `app.js`: local note saving, navigation, and CSV export
- `data.js`: generated sample data for the 50 review cases
- `assets/images/`: copied review images

## How to use

1. Open `index.html` in a browser.
2. Review the image and QA pair.
3. Enter the doctor's treatment advice in the text box.
4. Click `Download CSV` to export all notes.

The notes are also saved locally in the browser with `localStorage`.
